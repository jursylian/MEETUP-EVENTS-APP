# Events Platform

## Description
A frontend web application for browsing and filtering events.

The project includes event cards, category filtering, search, date-related information, and online/offline event views. It focuses on building a responsive and user-friendly interface for event discovery.

## Features
- Event listing with cards
- Search functionality
- Category-based filtering
- Online / offline event views
- Date display for events
- Responsive design for different screen sizes

## Tech Stack
- HTML
- CSS / SCSS
- JavaScript

## What I learned
- Building responsive interfaces
- Implementing filtering and search logic
- Structuring interactive frontend pages
- Improving usability and visual hierarchy
